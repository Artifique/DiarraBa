🎯 PROMPT DE GÉNÉRATION D'INTERFACE
Design a modern, premium farm management web application called "Diarraba Volailles" 
built with Next.js 14 (App Router), Tailwind CSS, shadcn/ui, and Framer Motion.

GLOBAL DESIGN LANGUAGE
━━━━━━━━━━━━━━━━━━━━━━━━━━━
- Dark theme with deep navy base (#0F1D4E)
- Glassmorphism cards: backdrop-blur + semi-transparent navy surfaces
- Orange (#F5A623) as THE accent — buttons, badges, highlights, icons
- Subtle mesh gradient backgrounds: radial gradients mixing navy + dark teal
- Grain texture overlay (SVG noise) at 3% opacity for depth
- Warm orange glow effects on hover states and active elements
- Green (#2D6A2D) used only for "success", "available", "paid" statuses
- Smooth page transitions with Framer Motion (fade + slide up, 300ms)
- Micro-interactions: card lifts on hover (translateY -4px + shadow)
- Consistent 8px border radius on cards, 6px on buttons

ANIMATIONS & POLISH
━━━━━━━━━━━━━━━━━━━━━━━━━━━
- Page load: staggered card entrance (each card fades+slides up with 80ms delay)

- Charts: draw-on animation (Recharts animationDuration=1500)
- Status badge changes: pulse animation (1 pulse when status updates)

- Orange glow pulse on notification badge when new notification arrives
- Loading skeleton: animated shimmer in navy tones (not white)
- Toast notifications: slide from top-right, auto-dismiss 4s (react-hot-toast)

💡 Conseils d'implémentation prioritaires

Commencez par le layout (sidebar + header + main) avec les tokens de couleurs
Dashboard en premier — c'est la vitrine, ça valide tout le design system
Réservations ensuite — c'est la page la plus utilisée
Utilisez shadcn/ui npx shadcn@latest init avec le thème dark comme base puis surchargez vers le bleu marine
Supabase RLS dès le début pour sécuriser par rôle (manager vs client vs fournisseur)
